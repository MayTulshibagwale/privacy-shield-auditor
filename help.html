🔍 Privacy Shield Auditor — Help Guide

Everything you need to know about what this extension does and what your score means.

📋 What Does This Extension Check?

Every time you open the extension on a website, it sends the page URL to a local Python backend which scans the site and returns three signals:

🔒

HTTPS Security

Checks if the site uses a secure, encrypted connection (HTTPS) or an unsafe one (HTTP).

⚖️

Policy Links

Scans the page for links to Privacy Policies, Cookie Notices, or Terms of Service.

🍪

Tracking Cookies

Counts cookies the site drops and flags any that belong to known third-party ad trackers.

📖 Glossary — Plain English Definitions

🍪 Cookie

A tiny text file a website saves on your browser. Some are useful (like remembering you're logged in), others track your behavior across the internet for advertising.

🎯 Tracker / Third-Party Tracker

A tracking cookie planted not by the site you're visiting, but by an outside company like Google, Facebook, or LinkedIn. These companies track you across many websites to build an ad profile.

🏠 First-Party Cookie

A cookie set by the website you're actually visiting (e.g. github.com sets a github.com cookie). Usually harmless — used for login sessions and preferences.

🌍 Third-Party Cookie

A cookie set by a different domain than the one you're on. For example, visiting news.com but getting a doubleclick.net cookie. This is how cross-site tracking works.

🔒 HTTPS

HyperText Transfer Protocol Secure. When a site uses HTTPS, all data between your browser and the server is encrypted. If a site only uses HTTP (no S), anyone on the same network can read what you send.

⚖️ Policy Link / Compliance Link

A link on the page that leads to a Privacy Policy, Cookie Policy, or Terms of Service. Laws like GDPR (Europe) and CCPA (California) require websites to have these if they collect data.

📊 Compliance Grade

A letter grade (A through F) our tool calculates based on the combination of HTTPS status, number of cookies, number of recognized trackers, and presence of policy links.
